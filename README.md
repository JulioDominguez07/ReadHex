# ReadHex
## Description
When given a file, it reads the content in binary and then in hex.
## Installation
1. Clone the repository
```
git clone https://github.com/JulioDominguez07/ReadHex
```
2. Compile the program
```
make
```
```
gcc -o readhex readhex.c
```
3. Use the program
```
./readhex foe.txt
```
